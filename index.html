Completed Projects

Dual Punk

Dual Punk is a rogue-like 2D shooter video game inspired by Hades and CyberPunk. It was made in a team of 5 as a school project in Epita. It uses the Unity game engine and is playable either alone or with a second player online.

Word Search Solver

This is a graphical application which can automically solve Word Search puzzles. It was made in a team of 4 as a school project in Epita. It uses OCR AI as well as image analysis techniques to recognize the letters in the image. The AI as well as it's training algorithm was implemented in C without the use of any libraries.

Noupharen

Noupharen is a graphical application used to generate and store dictionaries of words based on user-defined rules. It was made as a personal project over a month, using the QT library. The purpose of this program is to facilitate the creation of conlangs, which are fictional languages.
